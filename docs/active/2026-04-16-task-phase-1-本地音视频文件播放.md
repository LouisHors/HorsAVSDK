## 任务描述
实现基于 FFmpeg 的本地音视频文件播放功能，支持 MP4/MOV/MKV/AVI 格式，H264/H265 视频解码（软解），AAC/MP3 音频解码，Metal 视频渲染。以 macOS 为验证平台。

## 遇到的问题
1. **FFmpeg 6.x API 兼容性**: `AVCodecContext.channels` 已弃用，需要使用 `ch_layout.nb_channels`
2. **H264 解码需要 extradata**: 解码器需要 SPS/PPS 数据才能正确解码，这在后续集成中需要从 demuxer 传递
